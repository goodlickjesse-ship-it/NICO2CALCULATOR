# NICO CALCULATOR

A modern, scientific calculator built with Jetpack Compose.

## Features
- **Basic Arithmetic**: Addition, subtraction, multiplication, and division.
- **Advanced Scientific Functions**: sin, cos, tan, log, ln, square root, powers, and factorials.
- **Trigonometric Modes**: Toggle between Degree (DEG) and Radian (RAD) modes.
- **Chained Expressions**: Support for complex formulas like `sin(30) + cos(60)`.
- **Adaptive UI**: Collapsible scientific panel and responsive layout.
- **Modern Design**: Dark theme with high-contrast buttons and a scrollable display.

## Built With
- **Kotlin**: Primary programming language.
- **Jetpack Compose**: For the modern, declarative UI.
- **exp4j**: Powerful math expression parser for complex calculations.
- **Material 3**: For styling and components.

## Installation
